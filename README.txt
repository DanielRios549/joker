=> IMPORTANTE <=


A pasta de vídeos ("media/movies" e 'media/series) não são versionadas por poderem ser muito grandes,
elas serão criadas automaticamente quando algum filme e/ou série é adicionado.


=> Gulp <=


1 - Use a tarefa "bower" para pegar os componentes caso atualize-os pelo bower.

2 - Use a tarefa "prod" para compilar o CSS para produção, futuramente vai ser implementado para JS.

3 - Use a tarefa chamada "sass" para compilar os estilos do site.

4 - Use a tarefa chamada "watch" para fazer a tarefa "sass" automaticamente ao salvar um arquivo SASS.

5 - Vc pode utilizar a tafera "sync' para auto recarregar o navegador ao salvar um arquivo, mas não é obrigatório.

6 - Se quizer, customize a tarefa "default" para fazer tudo que vc quizer.
