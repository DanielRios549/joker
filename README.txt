=> IMPORTANTE <=


A pasta de vídeos ("media/movies" e 'media/series) não são versionadas por poderem ser muito grandes,
elas serão criadas automaticamente quando algum filme e/ou série é adicionado.


=> Gulp <=


0 - Use a tarefa "prod" apenas na branch master, para compilar o CSS e o JS para produção.

1 - Use a tarefa "dep" para pegar os componentes caso atualize-os pelo npm.

2 - Use a tarefa "sass" para compilar os estilos do site.

3 - Use a tarefa "watch" para fazer a tarefa "sass" automaticamente ao salvar um arquivo SASS.

4 - Vc pode utilizar a tafera "sync' para auto recarregar o navegador ao salvar um arquivo, mas não é obrigatório.

5 - Se quizer, customize a tarefa "default" para fazer tudo que vc quizer.
